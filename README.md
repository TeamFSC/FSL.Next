# FSL.Next

FSL的下一代版本！

FSL.Next具有更加**简易的配置模式**，

并且支持了**多账户管理**，不需要单独配置账户信息。

经历了种种体验优化，FSL.Next将FSL推上了新高度。

更重要的是，它看起来**非常好看**！

个人认为FSL.Next好看一点吧...其实FSL.RH和FSL.Next是截然不同的UI，
